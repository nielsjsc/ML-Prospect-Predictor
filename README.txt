# MLB Prospect Prediction Model

## Overview
This project uses machine learning to predict MLB prospects' future performance using data from minor league statistics and scouting reports.

## Features
- Prospect data processing and matching
- ML model for performance prediction
- Google Cloud integration
- Interactive UI for predictions

## Setup
1. Clone repository
2. Create virtual environment: `python -m venv venv`
3. Activate venv: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Unix/Mac)
4. Install requirements: `pip install -r requirements.txt`
